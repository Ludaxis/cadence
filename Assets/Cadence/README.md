# Cadence - Dynamic Difficulty Adjustment SDK

Game-agnostic DDA SDK for Unity. See the [full documentation](https://github.com/Ludaxis/cadence#readme) for installation, API reference, and examples.

## Modules

- **Signals** — Collect, buffer, persist, and replay gameplay signals
- **Analysis** — Session-level statistics with exponential running averages
- **Player Model** — Glicko-2 skill rating with confidence intervals
- **Flow Detection** — Real-time state classification (Flow, Boredom, Anxiety, Frustration)
- **Adjustment** — Rule-based difficulty proposals with cooldown and damping

## Requirements

- Unity 2021.3+
- No external dependencies
